
ΨΜΑ ΤΥΠΟΥ Ι, ΤΕΧΤ TEMPLATE - 21/7/2023

Είναι κείμενο που μπορεί να περιέχει και μορφοποίηση (και εικόνες) πχ. από word
Δεν ανεβάζουμε απευθείας το word αρχείο αλλά πρέπει να κάνουμε copy / paste 
μέσα στο template αφού πρώτα περάσουμε το κείμενο μας μέσα από ειδικό online editor
ώστε να διατηρηθεί τυχόν μορφοποίηση (bold, italics κτλ.)

Η χρήση των ΨΜΑ θα θεωρηθεί εμπορική, οπότε όλα τα resources (εικόνες, ήχοι, video, κείμενα, εφαρμογές, html κτλ.) πρέπει να είναι είτε
free, original, είτε με αγορασμένα δικαιώματα. Προσπαθούμε για χρήση όσο γίνεται free υλικού και όπου χρειάζεται άδεια το αναλαμβάνει ο όμιλος.


ΔΙΑΔΙΚΑΣΙΑ - ΒΗΜΑΤΑ

1) Ανοίγουμε στον browser μας το νέο online editor https://www.poukamisas.gr/ftp/editor
(ο προηγούμενος είχε όριο χαρακτήρων και δεν υποστήριζε διάφορα πράγματα πχ. εικόνες)

2) Εκεί είτε γράφουμε απευθείας κείμενο, είτε κάνουμε copy - paste έτοιμο κείμενο πχ. από word μέσα στον editor
Κατά το copy - paste οι εικόνες μετατρέπονται σε κώδικα αυτόματα και ενσωματώνονται μέσα στην HTML
ΝΟΤΕ: Ενδέχεται αν ξεπεραστεί κάποιο συνολικό μέγεθος (σε ενσωματωμένες εικόνες) οι εικόνες να μην ενσωματώνονται πλέον
μέσα στην HTML. Επειδή αυτός είναι ένας περιορισμός που δεν φαίνεται να μπορούμε να αποφύγουμε
Προτείνεται είτε να χρησιμοποιούνται μικρές - λίγες εικόνες ή να μετατρέπουμε λίγες σελίδες word κάθε φορά
Εναλλακτικά μπορούν να προστεθούν και εικόνες manually - χειροκίνητα από εμάς 
(βλ. το σχόλιο: "ΨΜΑ TODO: Ανανέωση κειμένου" μέσα στην "index.html")

3) Αφού κάνουμε paste μέσα στον editor και εμφανισθεί το περιεχόμενο μας, καλό είναι να το ελέγξουμε πχ. μπορεί κάποιες εικόνες 
να εμφανίζονται σε μεγαλύτερο μέγεθος (αν ναι, μπορούμε να κάνουμε κλικ επάνω τους και scale) ή όπως αναφέρθηκε κάποιες εικόνες να μην εμφανίζονται καθόλου 
(ή σαν missing images). Σε αυτή την περίπτωση πρέπει να κάνουμε copy paste μικρότερη ποσότητα κειμένου (κυρίως περιλαμβάνοντας λιγότερες εικόνες μέσα)  

4) Αφού είναι οκ το περιεχόμενο μας, κάνουμε click στο κουμπί "Κώδικας" του editor για να έχουμε πρόσβαση στην παραγόμενη html 
την οποία επιλέγουμε όλη πχ με Ctrl + A για να την κάνουμε paste μέσα σε συγκεκριμένο σημείο στην "index.html" (βλ. επόμενο βήμα)

5) Κάνουμε edit την "index.html" (που βρίσκεται μέσα στο template folder) για να αλλάξουμε τα στοιχεία πχ. τίτλος ΨΜΑ, κυρίως κείμενο, 
κείμενο βοήθειας-πληροφορίες κτλ. 
Βλέπε comments μέσα στην index.html που περιέχουν τη λέξη "ΨΜΑ"  πχ. "ΨΜΑ TODO: Ανανέωση τίτλου"
Ειδικά για το κείμενο εντοπίζουμε το comment "ΨΜΑ TODO: Ανανέωση κειμένου" εδώ θα χρειαστεί να αντικαταστήσουμε το υπάρχον κείμενο,
με το κείμενο (κώδικα html) που κάναμε copy από τον editor (το φέρνουμε μέσα με copy - paste)
[Για το edit χρησιμοποιούμε πχ το notepad++ (notepad-plus-plus.org) ή το sublime (sublimetext.com) ή οποιοδήποτε άλλο html editor]

6) Σώζουμε τις αλλαγές μας (κάνουμε save το αρχείο "index.html")

Τέλος πρέπει να ανανεωθούν και τα στοιχεία μέσα στη σελίδα συντελεστών "about.html" (είναι μια σελίδα με τους συντελεστές δηλ. τίτλος ΨΜΑ, ημ/νίες, συντελεστές) ενδέχεται να μην είναι αυτή η τελική της μορφή (έχει ζητηθεί διευκρίνιση)
